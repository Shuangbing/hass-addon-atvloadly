# ATVLoadly Add-on

ATVLoadly add-on for Home Assistant.

## About

This add-on provides ATVLoadly functionality for Apple TV.

## Installation

1. Add this repository to your Home Assistant instance
2. Install the ATVLoadly add-on
3. Start the add-on

## Configuration

No additional configuration required.

## Usage

The add-on will run on port 5533.
